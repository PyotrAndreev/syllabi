# 201 High Load Python

Do you know how to make Python use the full power of modern hardware?  
In this course, we go down to the silicon level to understand how CPUs and GPUs deliver maximum performance:
- cores, caches, memory bandwidth, SIMD, parallel execution, and data movement  

Then we connect that hardware view to Python under the hood: JIT, asyncio, parallelism, and accelerator-backed execution. We test theory with benchmarks and build a practical decision map for choosing the right acceleration path in production.  

__Target audience__: Middle Python devs who want to go deeper into how code is actually executed — from Python internals to CPU/GPU hardware.  
It is aimed at people who want to explore execution models, parallelism, async, JIT, and hardware-aware performance.

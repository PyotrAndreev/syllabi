This course will help you prepare for the production world and strengthen your portfolio through real projects.# 101 Production-Grade Python

Do you know how to turn Python code into a production system?  
`In 101 Production-Grade Python`, you’ll learn how to:
- design, build, test, package, deploy, and debug real Python services.  

We will look at modern frameworks that are responsible for:  
- APIs, databases, brokers, Docker, CI/CD and system design  

You will see the whole architecture fits together, learn how to explain your technical decisions, and build your own project.  
This course will help you prepare for the production world and strengthen your portfolio through real projects.  

__Target audience__: juniors, middles Python devs who want to move from “it works on my machine” to “I can ship, scale, and support it.”
